Installing Tor:

  ~$ sudo pacman -Sy tor

Enabling Tor:

  ~$ sudo systemctl start tor; systemctl status tor

Configuring Proxychains4:

  Within the /etc/proxychains.conf directory
  you'll need to ensure the following.

  1. Ensure that dynamic chains are set.
  2. Ensure that dns is being proxy routed.
  3. Ensure that socks5 is used.

Using tor with proxychains:
