PPA - Personal Package Archive

This is a software repository for uploading source packages to be
built and published.

They're used as an APT repository entry by canonicals launchpad service.

Anyone could make a PPA which means they're not very secure; they could
be easily used to ditribute melicious code.

You can ADD a PPA through these commands:
1. sudo apt-get-repository ppa:<repository>
2. sudo apt-get update
3. sudo apt-get install <name of application from repository>

There's four kinds of repository:
1. Main - Canonical Supported FOSS repositories.
2. Universe - Community maintained FOSS repositories.
3. Restricted - Proprietary drivers.
4. Multiverse - software restricted by copyright issues.

You can see these sources in /etc/apt/ in the source.d file or the source.d
directory therein.
